Detailed Description

A basic field Versit property.

The structure of Versit property is defined in VBK_FIELD_VERSIT_PROPERTY in VPbkFieldType.rh.

See also: MVPbkFieldType MVPbkFieldType::Matches

library
VPbkEng.lib
