RConnection API Summary Reference

This topic provides a summary of the functionality of the RConnection class.

System wide connection management and monitoring

Functions that apply to all connections, not just the current one.

  • RConnection:EnumerateConnections(TUint&)

  • RConnection::AllInterfaceNotification(TDes8&, TRequestStatus&)

  • RConnection::CancelAllInterfaceNotification()

Sub-Connection management and monitoring APIs in RConnection

Retrieves information on RSubConnection s and manages RSubConnection s.

  • RConnection::Stop(TSubConnectionUniqueId)

  • RConnection::Stop(TSubConnectionUniqueId, TConnStopType)

  • RConnection::ProgressNotification(TSubConnectionUniqueId, TNifProgressBuf&, TRequestStatus&, TUint)

  • RConnection::CancelProgressNotification(TSubConnectionUniqueId)

  • RConnection::Progress(TSubConnectionUniqueId, TNifProgress&)

  • RConnection::IsSubConnectionActiveRequest()

  • RConnection::IsSubConnectionActiveCancel()

Connection settings for RConnection

Allows the configuration settings for this connection to be retrieved from CommsDat.

  • RConnection::GetIntSetting(const TDesC&, TUint32&)

  • RConnection::GetBoolSetting(const TDesC&, TBool&)

  • RConnection::GetDesSetting(const TDesC&, TDes8&);

  • RConnection::GetDesSetting(const TDesC&, TDes16&);

  • RConnection::GetLongDesSetting(const TDesC& aSettingName, TDes& aValue)

Data monitoring APIs in RConnection

Gets statistics on data throughput.

  • RConnection::DataTransferredRequest(TPckg<TUint>&, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataTransferredRequest(TSubConnectionUniqueId, TPckg<TUint>&, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataTransferredCancel()

  • RConnection::DataTransferredCancel(TSubConnectionUniqueId)

  • RConnection::DataSentNotificationRequest(TUint, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataSentNotificationRequest(TSubConnectionUniqueId, TUint, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataSentNotificationCancel()

  • RConnection::DataSentNotificationCancel(TSubConnectionUniqueId)

  • RConnection:: NotificationRequest(TUint, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataReceivedNotificationRequest(TSubConnectionUniqueId, TUint, TPckg<TUint>&, TRequestStatus&)

  • RConnection::DataReceivedNotificationCancel()

  • RConnection::DataReceivedNotificationCancel(TSubConnectionUniqueId)
